# 🚀 n8n Workflow Community Hub

This repository contains a collection of useful and ready-to-use **n8n workflows** shared by **jbjardine**, made available for the n8n community.

Feel free to explore, reuse, and adapt these workflows to boost your automation projects!

---

## 📁 Available Workflows

- **[n8n Custom Auth Credential Updater](./update_credential_custom_auth)**  
  Easily and securely update your n8n Custom Auth credentials.  
  ⚠️ **Note**: Currently tested and approved **only** for self-hosted n8n instances.

*(More workflows coming soon!)*

## 📖 How to use

Each workflow is stored in a dedicated folder, along with detailed documentation for quick setup and use.

- **Browse**: Explore each workflow folder.
- **Download**: Import workflows directly into your n8n instance.
